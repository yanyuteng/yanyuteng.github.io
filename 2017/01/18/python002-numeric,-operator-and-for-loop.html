Lecture 7 打飞机的游戏机制

原理：进入一个大循环，只要未触发死亡，小飞机不断生成。
1、加载背景音乐
2、播放背景音乐（循环）
3、我方飞机生成
4（5）、interval = 0 #没有interval会不间断生成满屏幕飞机
5（4）、进入循环 

while True：  
    if 用户是否点击了关闭按钮：  
          结束程序  
    else  
          继续  

    interval += 1 #累计时间以刷新飞机

       if interval == 50：
          interval = 0 #刷新interval时间
          小飞机诞生

     小飞机移动一个位置 #移到if外 否则会卡顿
     屏幕刷新

     if 用户鼠标产生移动：
        我方飞机中心位置=用户鼠标位置
        屏幕刷新

     if 我方飞机与小飞机肢体冲突：
        我方死亡，播放死亡音乐
        我方飞机图案修改
        打印’game over’
        渐变停止背景音乐

Lecture 8 条件表达式与assert

一、输入转换

1、方法一：常规法

score = int(input('please write down the mark:'))
if 100>= score >= 90:
    print('A')
if 90> score >= 80: #二次后的判断是浪费的
    print('B')
if 80> score >= 60:
    print('C')
if 60> score >= 0:
    print('D')
if score <0 or score >100:
    print('false')

2、方法二：但缩进麻烦

score = int(input("please write down the mark:"))
if 100>=score>=90:
     print('A')
else:
     if 90> score >= 80:
          print('B')
     else:
          if 80> score >= 60:
               print('C')
          else:
               if 60> score >= 0:
                print('D')
           else:
                print('false')

3、方法三：改进二

score = int(input('please write down the mark:'))
if 100>= score >= 90:
     print('A')
else 90> score >= 80: #C else if 节省判断时间
     print('B')
else 80> score >= 60:
     print('C')
else 60> score >= 0:
     print('D')
else:
     print('false')

#python可以有效避免悬挂else
#C语言就近if else匹配，python则利用tab减少不确定性。

二、条件表达式（三元操作符）

1、含义：元为操作数，三元即是三个操作数。
2、例子

x, y = 4, 5
if x < y:
     small = x
else:
     small = y
small = x if x < y else y

三、断言（assert）

1、含义：当这个关键字后面条件为假时，程序自动终止，打印AssertionError异常。
2、作用：设置检查点，确保程序中某个条件为真才能让程序工作。
assert 3 > 4

一、for循环

1、语法
for 目标 in 表达式:
循环体
2、例子

for i in practice:
    print(i)
for i in practice:
    print(i, end="")

number=['yanyuteng','demos','xiaojiayu','python']

for i in number:

二、range()

1、语法
range([start,] stop[, step=1])
（1）[]内两个参数可选
（2）step=1默认为1
（3）生成从start值到stop值结束的数字序列
2、配合for

range(4) #range(0, 4)
list(range(4)) #[0, 1, 2, 3]
for i in range(5):
     print(i)
# 生成0到4
for i in range(2,9):
    print(i)
# 生成2到8

三、break与continue

1、break
含义：满足条件才可以终止循环，跳出循环体
2、例子

yes = "yanyuteng"
answer=input("the handsome man's name:")
while True:
    if answer == yes:
        break
    answer = input("sorry,u are wrong, please guess again:")

print('smart!')

2、continue
1、含义：终止本循环，开始下一循环
2、例子

for i in range(10):
    if i%2 != 0:
        print(i)
        continue
    i += 1
    print(i)
