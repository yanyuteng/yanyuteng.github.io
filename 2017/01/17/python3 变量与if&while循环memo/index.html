Lecture 3 变量

一、变量

variable存储于内存中，给变量赋值。python没有变量，只有名字，更多地是将名字贴到值上面。
0、数值

a = 1
b = 2
c = a+b
print(c)

1、字符串（文本）

a = "b"
print(a)
a = “c"
print(a)

# 字符串的拼接 bc

a = "b"
b = "c"
c = a+b
print(c)

二、字符串（文本）

0、字符串两侧有一致引号
1、字符串中需要出现单引号
（0）转义符号 \

print('Let\'s go!')

（1）双引号套单引号

print("Let's go!")

2、原始字符串
（0）输出c:\now

str = "c:\now"
str #输出为'c:\now’
print(str) #false /n被print认为是换行

（1）c:\now被print函数识别

str = "c:\\now” #以\转义\
print(str) #ok

（2）多个\：考虑原始字符串r 

#如果有很多\则不能依次转义，会浪费资源，考虑原始字符串，在字符串前加r
str = r"c:\now\a\b”
str #输出为'c:\\now\\a\\b'
print(str)

（3）字尾加\

str = r"c:\now\a\b\” #false 如何字尾加\
str = r"c:\now\a\b"
c = "\\"
print(str+c)

（4）长字符数传

#跨越多行字符串时，考虑三重引号字符串 '''…'''
str='''我刚才读Yeats的诗歌，
有些话很有共鸣，
一个青年对情人的占有欲'''

Lecture 4 猜数字游戏改进

一、条件分支：if else

0、目的：用户猜错后，程序提示大或小
1、比较操作符

> #左面大于右面 eg.1 > 3 False  
>= #左面大于等于右面  
== #等于  
!= #不等于
2、改进 temp = input("guess my number:") guess = int(temp) #define guess if guess == 8: print("yes") else: if guess > 8: print("u should guess smaller") else: print("u should guess bigger") print(“amazing, end")

二、while循环：重复猜测

0、目的：多次输入，也即重复运行代码
1、改进

temp = input("guess my number:")
guess = int(temp) #define guess
while guess != 8:
     temp = input(“wrong, please try again:")
     guess = int(temp)
     if guess == 8:
         print("yes")
       else:
         if guess > 8:
            print("u should guess smaller")
         else:
             print("u should guess bigger")
print("amazing, end")</code></pre>

三、and逻辑符：限制猜测次数

0、and 与逻辑操作符
可以将任意表达式连接到一起，得到一个布尔类型(TRUE&FALSE)值
1、符的优先级
3>2 and 1<2
3<2 and 1<2
#比较运算符优先级大于逻辑操作符，良好习惯可以加()优先执行
2、改进 

temp = input("guess my number:")  
guess = int(temp) #define guess  
i=1  
while guess != 8 and i<=2:  
     temp = input("wrong, please try again:")  
     guess = int(temp)  
     if guess == 8:  
         print("yes")  
         print("it's amazing.")  
     else:  
         if guess > 8:  
             print("u should guess smaller")  
         else:  
             print("u should guess bigger")  
         i=i+1  
         print("unfortunately.")  
print("thanking u for ur gussing")</code></pre>

四、random模块：每次猜测的答案随机

0、引入random模块的randint()：返回随机整数
1、随机数改进

import random  
secret= random.randint(1,10)  
temp = input("guess my number:")  
guess = int(temp) #define guess  
i=1  
while guess != secret and i<=2:  
     temp = input("wrong, please try again:")  
     guess = int(temp)  
     if guess == secret:  
         print("yes")  
         print("it's amazing.")  
     else:  
         if guess > secret:  
                 print("u should guess smaller")  
         else:  
                 print("u should guess bigger")  
    i=i+1  
    print("unfortunately.")  
print("thanking u for ur gussing")</code></pre>

2、进一步改进：第一次也有提示大或小

import random
secret= random.randint(1,10)
temp = input("guess my number:")
guess = int(temp) #define guess
if guess == secret:  
      print("yes")  
      print("it's amazing.")  
else:  
    if guess > secret:  
        print("u should guess smaller")  
    else:  
        print("u should guess bigger")  

i=1 # first try over

while guess != secret and i<=2:  
    temp = input("wrong, please try again:")  
    guess = int(temp)  
    if guess == secret:  
        print("yes")  
        print("it's amazing.")  
     else:
        if guess > secret:
                print("u should guess smaller")
        else:
                print("u should guess bigger")
    i=i+1 # i+1 try
    print("unfortunately.")
print("thanking u for ur gussing")
