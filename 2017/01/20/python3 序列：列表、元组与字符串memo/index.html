Lecture 10 11 list基础

一、list含义

1、数组即同种类型数据放在一起。python没有数组，而以列表代替。
2、列表可以将对象、整数、浮点数、字符串等加入，基本所有的python都需要列表。

二、list创建

member = ['a', 'b', 'c', 'd']
number = [1, 2, 3, 4]
mix = [1, 'a', 3.14, [1,2,3] ]
empty = []

三、list添加元素

1、append() 尾部加入单个

member.append('e')
len(member)

2、extend() 尾部加入列表

# 使用一个列表扩展另一个列表，其参数是一个列表
member.extend(['e’,'f'])

3、insert() 加入任意位置

member.insert(0,'g')
# 列表索引值从0开始

四、list调换元素：调换a、b位置

member=['b','a','c','d']
temp=member[0]
member[0]=member[1]
member[1]=temp

五、list删除元素

1、remove()

member.remove('d')
# 必须知道名字，不需知道顺序

2、del语句

del member[1]
# 知道顺序

3、pop()

name = member.pop()
#默认取出最后一个元素，并显示已取出
member.pop(1)

六、读取多个元素：列表分片（slice）

利用索引值，每次可以从列表中获取一个元素，如何一次性获取多个元素。 

member = ['a', 'b', 'c', 'd']
member[1:3]

Lecture 12 list操作符与BIF

一、list中常用操作符

1、比较操作符

list1 = [123,456]
list2 = [234,123]
list1>list2 #FALSE 比较第0个

2、逻辑操作符

list3 = list1
(list1 < list2) and (list1 == list3)

3、连接操作符

list4 = list1 + list2
list4 #[123, 456, 234, 123]
list1 + ‘yyt' #FALSE +号两边需要一致
# 此处应该用extend或append更规范

4、重复操作符

list3* = 3

5、成员关系操作符

123 in list3
'yyt' not in list3
list5 = [123, ['yyt','demos'], 456]
'yyt' in list5 #FALSE 只判断一层
'yyt' in list5[1]
list5[1][1] #demos

二、list的相关函数

dir(list) #列表相关BIF

1、count()

list5.count(123) #返回出现次数

2、index()

list5.index(123) #返回位置
list5.index(123，3，7) #返回3到7中 123的位置

3、reverse()

list5.reverse() #翻转list5顺序

4、sort()

sort(func,key,reverse)
list6 = [4,3,2,5,9,10,99,0,-1,8]
list6.sort() #从小到大排序
list6.sort(reverse = True) #从小到大排序

5、list[:]分片

list6 = [99, 10, 9, 8, 5, 4, 3, 2, 0, -1]
list7 = list6[:] #分片拷贝：完全复制后保存独立性
list8 = list6 #不完全复制，跟着list6变化而变化

Lecture 13 元组 tuple

一、创建和访问

# 元组一旦定义不可改变，戴了枷锁的list
# 创建元组大部分用()，但根本区分在于,
apple = (1,2,3,4,5,6,7,8)
apple[1]
apple[5:]
8 * (8) # 64
8 * (8,) # (8, 8, 8, 8, 8, 8, 8, 8)

二、更新和删除

1、更新

temp = ('a', 'b', 'c', 'd')
temp = temp[:2] + (‘e',) + temp[2:]

2、删除

del temp # 完全删除 较少使用

三、操作符同于list

+, *, in not in, > < =, and or...

Lecture 14 字符串BIF

# 字符串同元组类似，一旦定义较难修改
str1 = "yan yu teng"
str1 = str1[:3] + ' str2' + str1[3:]

一、转换类

1、大小写

# capitalize() 第一个字母大写
str1.capitalize() #'Yan str2 yu teng'

# casefold() lower() 所有字符改为小写
str2 = 'YAN YU TENG'
str2.casefold() #'yan yu teng'
# str1.upper() 所有字符改为大写

# swapcase() 大小写翻转
# title() 返回title样式

2、格式对齐

# center(width) 将字符串居中，并用空格填充字符串左右width长度新字符串
str2.center(30) #'         YAN YU TENG          '
# str1.ljust(width) 将字符串左对齐，并用空格填充字符串右width长度新字符

# strip(chars) 去除左右空格，chars为可选同时去掉的字符
# lstrip() 去除左面空格

# expandtabs(tabsize = 8) 将字符串中tab（\t）转换为空格，默认为8
str3 = '    yanyuteng’ #'\tyanyuteng'
str3 = str3.expandtabs() #'        yanyuteng'

3、插入或分割

#join() 将字符串均匀插入到sub中（作为分隔符）
str1.join('123')#'1yan str2 yu teng2yan str2 yu teng3'

# partition(sub) 找到sub后分割为三个元组，不包含则返回(‘xxx' , , )
str1 = 'yan yu teng'
str1.partition('yu’) # ('yan ', 'yu', ' teng')

#split(sep = ’none’, maxsplit = -1) 自动切片为列表，默认以空格为切割

4、替换类

# replace(old,new,count) new替换old，次数为count
# translate(str.maketrans()) 按ASCLL码转换
str1.translate(str.maketrans('y','d’)) # 'dan du teng'
# encode(encoding = 'utf-8', errors = 'strict’) 以encoding指定编码格式编码

二、查找类

# count(sub,start,end) 返回sub在字符串中的出现次数，start和end为可选范围点
str2.count('Y’) #2

# find(sub,start,end) 查找sub是否包含在字符串中，有则返回第一个元素索引值，没有返回-1
# rfind(sub,start,end) 右侧开始查找
# index(sub,start,end) 同上，没有返回异常

三、判断类

# endswith(sub,start,end) 检查字符串是否以sub字符串结束，start和end为可选范围点
# startswith(sub,start,end)
str1.endswith('g’) #True

# isalnum() 字符串至少有一个字符，且全为数字或字母返回True，否则返回False
# isalpha 字符串至少有一个字符，且只包含字母字符返回True，否则返回False
# isnumeric() 字符串至少有一个字符，且只包含数字字符返回True，否则返回False

# isdligit() 只含数字返回True，否则返回False
# isspace() 判断只含空格
# istitle() 判断单词首字母大写，后面小写（标题化）

Lecture 15 字符串的格式化

格式化目的：格式化字符串，按照统一的规格输出，如果规格不统一，如16进制1 0与10进制1 0。

一、format方法

format方法接受位置参数和关键字参数两种，二者传递到replacement字段（花括号）。
1、位置参数

"{0} love {1}.{2}".format('yan','yu','teng','demos’) #'yan love yu.teng'

2、关键字参数

"{a} love {b}.{c}".format(a='yan',b='yu',c='teng')

3、混合使用

"{0} love {b}.{c}".format('yan',b='yu',c='teng')
# 位置参数必须在关键字参数之前，0必须在b c前，否则报错

4、格式化符号 :

{{0}}.format{'不打印’} #没有字段可以被输出，0已经被解释了，同/一样
'{0: .1f}{1}'.format(27.658,'GB’) #位置参数多了:
# :表示格式化符号的开始；.1四舍五入保留一位；f为定点数，同浮点数类似，打印一个小数。

二、字符串格式化符号含义

# %遇到字符串时变为格式化符号
%c 格式化字符及其ASCII码
'%c' % 97 # 'a'
'%c' '%c' '%c' % (97, 98, 99) # 'abc'

%s 格式化字符串
'%s' % 'yan yu teng’ # 'yan yu teng'

%d 格式化整数
'%d + %d = %d' % (4, 5, 4+5)

%o 格式化无符号八进制数 # 0-7 00=8 01=9
%x 格式化无符号十六进制数 # 0-9 a=10 b=11 c=12 g=16 00=17 a0=160
%X 格式化无符号十六进制数（大写）
'%x' % 10 # 'a'
'%X' % 10 # 'A'

%f 格式化定点数，可指定精度
'%f + x =' % 27.658 # '27.658000 + x ='
%e 科学计数法保留
%g 智能根据值大小决定用%f %e
'%g' % 27.658 # '27.658'

三、格式化操作辅助命令

1、m.n # m最小总宽度，n小数点后位数

‘%.2e' % 27.658 # '2.77e+01’ (2.77*10^1)

2、- +

- # 左对齐
+ # 正数前面显示+，负数前- （打酱油符号）

3、# 转换八进制前用(‘0o’)；转换十六进制前用(‘0x’)或(‘0X’)

'%#o' % 8 # ‘0o10' 八进制的10
'%#X' % 108 # ‘0X6C’ 十六进制的6C

4、0 #前面都用0填充

'%010d' % 5 # '0000000005'
'%-010d' % 5 # '5         '

四、字符串转义符含义

\n 换行符
\t 横向制表符
...

Lecture 16 序列

一、列表、元组和字符串共同点 ：序列

1、可以通过索引得到每一个元素
2、默认索引值从0开始
3、分片方法得到一个范围内元素集合
4、很多共同的操作符
# 列表、元组和字符串统称为序列

二、序列的相关BIF

list() # 不带参数空列表或带一个迭代器作为参数
# 迭代 重复反馈过程的活动，其目的是为了接近或达到接近的目标，每一次迭代的结果作为下一次迭代的初始值。序列本身是可迭代对象，迭代本身是序列泛化而来。

1、转换

tuple([iterable]) # 可迭代对象转换为元组
str(obj) #把obj对象转换为字符串
 c=(1,1,2,3,5,8,13,21,34)
 c = list(c) # [1, 1, 2, 3, 5, 8, 13, 21, 34]

2、计算

len(sub) #返回sub的长度

min(sub)
max(sub) #返回序列或参数集合中最大值，字母则返回ASCLL码最大值
max() 迭代原理：
 max = tuple(0)
 for i in tuple:
     if i > max:
         max = i
 return max

sum(iterable[,start=0]) #加总

3、排序

sorted() #默认从小到大排序 sorted(numbers)
reversed() #返回迭代器对象 list(reversed(numbers))从大到小排序

4、拼接

enumerate() #返回（index, item)的元组
number = (2,1,3,4)
list(enumerate(number))

zip #返回成对参数打包的元组
a = [1,2,3,4,5,6,7,8]
b = [4,5,6,7,8]
list(zip(a,b)) #[(1, 4), (2, 5), (3, 6), (4, 7), (5, 8)]
