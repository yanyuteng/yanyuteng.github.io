日记 不如刷牙

发表于 2015-09-07   |   分类于 挣扎集   |     |   本文總閱讀量 次

多少是有些郁郁，写些字来消遣，从八点半到现在。
猛然便发现自己不再是freshman了，恍惚已是站在了十八岁的尾巴，下个月就要19岁了。转眼当年的孩子已经长大，从垂髫少年，变成了今天的弱冠。
十几年匆匆而过，回头看，好多东西都记不清澈，模模糊糊。
大一这年，去了蛮多地方，一个个城市与农村间腾挪。

但从兴城回来后，或是说从延边回来后，便有点颓废，对旅行似乎不再有太多的期待，对未来亦是深切的迷惘。在家蜗居了两天，懒得出门，看看书，愣愣神，一天奢侈的度过。
可能是有些压抑，这些年越来越不像自己，好多世俗的东西，韦伯说的“铁笼”。
愈发的喜欢听那些民谣，喜欢牛头琴的声音，喜欢吉他，喜欢从一个民谣到另一个民谣，像是寻找新大陆一样，找到那些小众的歌手。有些共鸣，然后更加孤独。
昨天看欧阳修的《梅圣俞诗集序》，“诗人少达而多穷。”穷而后工，大概好多的民谣人，也被慢慢遗忘在了角落，像尹吾，像我们。

不再听太多，原先爱听的柔软流行。
“我们都只剩一堆用青春编织的回忆。”
“云层深处的黑暗啊，淹没心底的景观。”
“一个人的路上只是在找寻。”
万能青年旅店里的南方姑娘呵，工体东路没有人。
何必去找。

刚才读到拜伦的《我愿做无忧无虑的小孩》里，“我不恨人类，只是想避开。”便蓦然想起了太宰治的《人间失格》，大概他们的落寞，些许孤独，拜伦战死，太宰治自杀。还有那个胡子好长好长的海子，倔强的坐在西藏，面朝大海。
又何必去死。

只是那些理想的东西呵。
“事隔经年，我与你再遇，该何以向你致意，以泪水，以沉默。”一盏台灯，笔记本的蓝光幽幽。
大概这诗多少不全是写给情人，亦是给现在的自己。成长总是放弃了一些理想，沉默在大海里。就像鲁迅说的：“我决不是一个振臂一呼应者云集的英雄。”

写到这里，抓了一大把石榴大嚼。恰好音乐循环到了Viva la Vida，便高呼了一声，生命万岁。
多少，在这坚硬的钢铁森林里，我依旧倔强的有着自己小小的精神王国。
便已足够。
或许某天我将不再言说，但有些幻想大抵是深深的，牢不可破的扎在心底。
那些感动是世俗无法给予的。

现在颇像吴嘎嘎说的：“理想碎成渣子的18岁的一天，一切都在向无意义很平庸的方向过度，半导体还在响，我光着脊梁，可惜没有肌肉显摆，24小时音乐台一刻不停的播着广告，我总是下意识的以为那是个24小时广告台。”
但终归，依旧借用吴嘎嘎的一句话。
“我们没有理由因为被生活强奸而做了生活的婊子，生活多美啊，你只要把你的小鸡鸡插进土地，你也可以强奸地球，生活多美啊，我们要好好的活下去，一定！”
女生或许可以试试美国国旗的杆子。

心情好多了。
一个普通人，不如刷牙去。
