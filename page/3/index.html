《自由主义》笔记

发表于 2015-11-30   |   分类于 Review   |  

导言
政治经济学取代了新古典经济学，批判法学摧毁了自由主义法学，法兰克福学派推倒了启蒙运动的自由主义，自由主义似乎陷入了衰败。但近二十年市场经济与国家干预经济的争论成为了西方政治改革的中心，美国里根主义、英国撒切尔主义，即为古典自由主义的复苏，工党、社会民主党等等，全球化中自由化潮流似乎又再度兴起。
是书即为对近来世界热议、国内争论的自由主义的梳理，作其历史、理论及他派批评的梳理。
极端而言，西方只有一种意识形态，即为自由主义，而保守与激进都是以自由主义为基础，手段与自由主义不同。而自由主义中，诸多主张并非一己一派的要要求，而大抵是现代社会的时代要求，即如市场经济，并非单纯的为资本主义特有。
由外及内，自由主义于中国不可避免涉及到中国传统是否亦有自由主义，于此如严复及孙中山持有的不同观点，便又涉及到了对自由主义意涵的界定。但终究诸多外延可独立成册，是书目的便重于探寻西方近现代自由主义的理论内涵。
是书的结构为三部分构成。
首先厘清自由主义的理论历史，以哈耶克等二战后自由主义为界定，英国、欧陆两种解释及参校英美传统。
继而从政治哲学范畴剖析内在统一原则，也即道德哲学在政治领域的应用与延伸。
第三部分，介绍以马克思主义、保守主义（古典自由主义与真正保守主义这两派的后者）、社群主义、左派思潮（近似于马克思主义）四个思潮，对自由主义的批评为主，后现代思潮对理性主义、普遍主义的批评因并未涉及自由主义的基本原则，而毋宁说是对其的发展，于此不述。
以上，是此导言提挈全书之要点。

《最近四十年中国社会思潮》 笔记

发表于 2015-11-28   |   分类于 Review   |  

写在随笔前面
是书多为他人资料，作者话语较少，作者之思考见解可为少年人理解当代社会思潮入门的简介，普及可矣，为一学术书籍则失真。
言语颇多新闻色彩，以时间顺序为主，各派人物穿插，人名繁多，一遍只能有些粗略印象，而无甚实质深入，此固积累过少的缘由，亦有些是书一些地方语焉不详，一些地方论述过细的原因，盖四十载风雨必难为廿万字洞见。
比较可惜的是于此版删去了旧版的各家评判，多少让此书失去了一些客观与深度，盖作者自由主义立场明显，论述左派时有明显的敌视意味，有些缺憾。
窥而生些感触，已矣。
读此书的顺序大概应先读其序（在思潮交锋中理解中国），再读其附录（最近四十年中国社会思潮的基本脉络），提纲挈领，尔后读其正文，如此可把握整体。
读此书源于近来颇感触，不理解当下社会思潮，便无法理解当下政治经济之走向，社会之现状，亦是发觉社会学生苟不稍加涉猎政治学内容，便难于深入社会现实。

日记 国际歌与些许社会学疑惑

发表于 2015-11-26   |   分类于 Prose   |  

2015.11.24
今日仿若突然明白了一件事情。
一直以来始终活于小我，前些日子读《近四十年中国社会思潮》时，对民众的不屑似乎更为明显，沾了知识的光亮，仿佛自己已是高人一等，而今日始觉康德因卢梭之转变，亦是于我个人之冲击。
知识的作用，在未能为大众真正谋得福利前，大抵不如劳动者，能直接创造价值的大众以实在的劳动对抗虚无，知识分子以思考对抗虚无。
而一国家终归不能消除，甚至助涨民族主义之情节，一人为人，当不为一教，不为一国，而为全人类而奋斗，如此回头看共产国际初创的崇高，高山仰止矣。
人类历史，亦确乎曾有那一时期，国际主义者为国际潮流，赤旗与自由主义的分野，于其根本出发点，又大抵无甚不同。
思至此，当初周恩来总理“为中华之崛起而读书”似乎又显得小气了，固然又不免有人要嘲笑我了，但真的猛士又岂为外人所动。
虽千万人吾往矣，何等的气魄，而这种转变也似乎让精神世界变得波澜壮阔。

晚上同班级合唱《国际歌》，固所愿也。

《昨日不辞而别》 随笔

发表于 2015-11-16   |   分类于 Review   |  

躁动与反抗：先摇滚乐发展侧记
文章以西安为主阵地，介绍八十年代摇滚乐，彼时摇滚渐为人知，崔健也还年轻。
九十年代，西安摇滚第一次黄金时期，拼盘是摇滚唱片的主要形式，恍惚记得我有一张2000后买的盘子，摇滚在艰苦与激情中反复。
世纪之交，西安摇滚的过渡与裂变。2000年前后，商业化的趋势与摇滚乐的坚守构成了很分明的对比，2001年八个半与绿洲。
03年，西安摇滚乐迎来了第二次黄金时期，地下演出与地下唱片勃兴，八十年代的精神延续。而读至此，我恍惚明白了18岁那年去西安，满街的吉他意味着什么，为何西安的摇滚昼夜不息，唱者街头比比，都是二十世纪不能复制的西安废都文化。
03年9月，八又二分之一的防空洞被政府收回，西安摇滚开始了短暂的无场地过渡。
04年，《掩灰的色彩：西安独立音乐合辑1》大卖，八个半短暂的停歇中，涌现的各酒吧与重开的八个半分庭抗礼。
而摇滚市场的不稳定，终归于06年后陷入了疲软期。
至09年，全国摇滚的疲软，酒吧、户外音乐节已不再被热捧。其间，西安摇滚以双喜的张冠李戴音乐节和月亮钥匙酒吧为中心。

中美政治制度的文化渊源 淺談

发表于 2015-10-29   |   分类于 Prose   |  

大一读帕累托《精英的兴衰》时，对其精英民主的构想，与勾勒出的阶层流动印象颇深。近来又读了些书，也愈发觉得中美政治制度，在精英统治上的共性，而在统治方式、统治精英的构成等方面的诸多差异，这个差异大抵更多的是源于文化，而不是源于经济。
一直以来，我笃信马克思主义在探讨资本主义产生及发展中，推崇的经济生产关系的决定作用，但近来亦是愈发接受了在经济之外，一个地区与国家，文化传统对制度的深刻影响，正如韦伯所推崇的多元因果论一般，经济绝不是单一的主导因素。

西方一直以来王权与神权的斗争，王朝政治贵族与修道院知识精英的斗争，初步构成了精英阶层的两分，即政治精英与知识精英。而之后，西方资本主义的发达，在韦伯的《新教伦理与资本主义精神》中曾有诸多探讨，大抵欧美资本主义的勃兴，有着有别于诸世界而被称为新教伦理的天职观与理性化因素，亦即后日衍化为推动科学反思的现代性。在新教徒之前，重商主义之下，经济精英便已日渐崛起，而新教在美洲大陆的扎根与崛起，更促成了今日美国经济精英的掌权。
在这种基督文化传统之下的西方，统治精英便不单为政治精英，而包含了由修道院中修士渐变而来的知识精英，和在天职观驱使下以挣钱为目的的经济精英。我思考，在今日西方形式上的三权分立之下，更深层次的依旧是这精英构成的三分，互相制约平衡。

你好，19岁

发表于 2015-10-03   |   分类于 Prose   |  

2号晚上，我和婧一姐一起看刚上映的《解救吾先生》，8点多吃饭，9点多打车回家，然后坐在书桌前，打开台灯，夜色深沉。
一直觉得到了2015年10月3日，十九岁开始的夜晚，应该写些东西来缅怀我这糊涂的十八岁，也曾浮想过很多看起来舒服动听的句子，闭眼细数十八岁一年的琐碎小事。

但执笔的时候，却发现墨水有些凝滞，久久没有下笔。
思绪万千，不知从何说起，大抵枕在寂寥的椅背上，有些东西是难以诉诸文字的，白首为功名，旧山松竹老。说我强说愁也罢，天凉好个秋也好。
只是前一阵读鲁迅先生给《呐喊》写的自序深而感触，“我决不是一个振臂一呼应者云集的英雄。”林语堂先生在《鲁迅之死》一文里也写过，“夫人生在世，所为何事？碌碌终日，而一旦瞑目，所可传者极渺。若投石击水，皱起一池春水，及其波静浪过，复平如镜，了无痕迹。”
十九岁，我终于知道，我只是个普通人。
