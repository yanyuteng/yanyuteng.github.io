譯 Immigrants and world cities

发表于 2016-03-16   |   分类于 Review_Sociology Papers

Marie Price,Lisa Benton-Short

作者核心論述的應該是反對過分著重經濟因素，在全球城市評定中忽視了移民因素，而整篇論文的經驗性研究就是圍繞論述移民與全球城市的相關性展開的。

一、導讀

研究城市是為了理解全球化，研究全球化在城市範圍的移民趨勢，也可以更好地理解大範圍移民如何創造一個新的移民目的地。
移民地有不同的形式，但是大多數是過度多樣化的，全球化連接了太多區域，並且在一些案例中，增長了很多新的隔離（移民）社區。
移民地通常是已經有了大量人口居住，觀察歷史，它們過去通常是城市中心並且吸引了多種族文化的人群，帝國和殖民地前哨站的貿易中心也通常是多樣化的，並且同時有著高度的隔離空間。

譯 Cities And Labour Immigration

发表于 2016-03-13   |   分类于 Review_Sociology Papers

Michael Alexander

對於較為文本核心的“THE HOST-STRANGER RELATIONS MODEL”概念,可以譯為“主客關係”、“熟悉人﹣陌生人”關係等，但仔細考慮后，我將之譯為了“本土﹣外來”關係。
因為是書所探討的，是城市與勞工移民，這其中就涉及到了原住居民與外來移民的互動，或者可以近似理解為台灣所謂的本省人和外省人，一種外來的進入如石子投入水中，蕩起水波。而類似翻譯為陌生人，則讓我想起了齊美爾的局內人局外人理論，對於理解是書有些干擾，譯為客也不是十分明確，所以最後譯為“本土﹣外來”關係。
此外，如“local policies toward migrants”翻譯為地方移民政策、“domains”翻譯為範圍或方面、“issue areas”翻譯為問題區等。

於此貼出了第一章的粗略翻譯內容，因為英文水平有限，一開始也沒有翻譯的打算，到後文逐漸熟悉作者意圖和反復出現的詞語后，覺得讀英文實在是蠻痛苦的事情，開始有了翻譯的意願，希望可以為其他同學提供一些參考。
前面的一些句子因為還不熟悉文本可能譯錯，如有錯誤還望指正原諒，此外，一些不重要的內容，在我認為不影響理解作者原意的基礎上也予以了捨去。

根據我的閱讀體驗，前面的敘述略微空洞，建議大家將重點放在作者對於類型學的介紹上（也即P13的3.2）、3.3中P15後半段的研究問題、3.4.2一節等。

《裸城》 The truman show

发表于 2016-03-10   |   分类于 Review_Sociology Books

導讀

AncientDemos
《裸城》的邏輯步步推進，每章前後的承接都頗為精彩，於此我就不再贅述裸城內容，而將具體各章心得，放在後面列出的讀書筆記以供參考。於此寫下這篇文章的目的，是想提出幾個問題，并嘗試將以往的閱讀與之結合，做一些自答與留待課堂繼續的討論。
頗為戲劇性的標題大抵是我仔細考慮后的選擇，標題源於電影《楚門的世界》，但之所以棄之未用，是因為它的英文名字或許更能體現我想表述的主旨，the truman show，the trucity show。
啟發于《裸城》的一句話，“他們共謀建構地方特色的表象和氣氛，以便自己體驗純正性。”印象里讀到這裡的震動，還頗為清晰。
他們共謀的建構，共謀的表演，以及共謀的自我欺騙，在城市空間里上演一個又一個的show，滑稽而又令人滿足的show，在城市空間里，true與show的分裂，暗合了純正性與其起源與開端。
一個時代的操縱性隱喻，與《裸城》不謀而合，帶給我們的是虛無或真實的迷茫與思考。

《巨變：當代政治與經濟的起源》memo

发表于 2016-03-09   |   分类于 Review_Sociology Books

導論

Fred Block
冷戰的結束，自由主義與功利主義的聯姻開始受人反思，由此催生了對於波瀾你著作的重新審視。
波蘭尼的家學淵源、流亡生涯、世界公民的人生經歷都頗為獨特。是書則是從制度面分析全球經濟，批評自由市場與我頗多共鳴。自由市場第一次巨變引發了法西斯主義的興起，亦埋伏了第二次巨變等。

經濟學所堅持的經濟體是自我調整的連鎖性市場背離了社會現實，而波蘭尼試圖指出人類經濟是嵌含在社會中，服膺于政治、社會關係。
古典經濟學希望建立自律性市場使社會服膺于市場，將社會關係嵌含在經濟體制中，而實際上這種脫嵌是一種空想，并必然不會存在。它之所以會是空想，在於市場的自主性與社會壓力之間的背離，人與自然環境成為商品，越商品化，社會越頻于解體，迫使經濟回歸。
土地、勞力、金錢並非市場生產，而經濟學將之等同於真實商品，將虛擬商品真實化，這意味著經濟學理論建構在謊言之上，將會把人類社會推向危機。
當政策傾向脫嵌，也即社會契約的削弱，政府亦將投入更多努力保證風險可控。
正是基於上述原因，波蘭尼由對經濟脫嵌的懷疑，而引申出了雙重動向理論，即市場自由放任擴張與保護主義的對立力量。
當自由放任與保守主義二者任何一個過強時，便會摧毀平衡，過度商業投機則虛假繁榮，泡沫破滅巨大打擊，政治經濟對立則如法西斯主義興起。
波蘭尼的獨特之處在於否定了自由市場資本主義與社會主義的區隔，社會主義試圖使自律市場服從于民主社會，工業文明的先天影響。

《偉大城市的誕生與衰亡》 筆記

发表于 2016-03-01   |   分类于 Review_Sociology Books

在讀《都市主義作為一種生活方式》的時候，猛然發現一些很重要的東西，似乎被我們習以為常的忽略了。
台灣大學的呂世浩先生曾在網路開設過《秦始皇》一課，其中有一個論點記憶尤深，中國迄今歷經了三次大變革，第一次為炎黃，第二次為始皇，第三次則是辛亥以來，這個觀點亦可在諸多學者的文章中看到。
呂先生等大抵是從制度或經濟的角度，而結合這兩篇文章，猛然意識到，人類生活方式的巨大轉變，未嘗不可亦作為巨變的標識。尤其是二十世紀以來城市化進程的巨大進展，在極短時間內從鄉村向城市轉移了大量人口，對人類社會產生了深遠的影響。

《偉大城市的誕生與衰亡》我是直接從第二部分看起，並未直接涉及都市的定義問題，所以借用沃斯在《都市作為一種生活方式》一文中對於都市的社會學定義在這裡就顯得頗為重要，對後續串聯兩篇文本起著參照點的連接作用，也即我試圖以沃斯之文本為基礎，比較雅各布斯之文本于其基礎上的異同。
但雅各布斯最關鍵的幾個論述，都是針對對芒福德等花園理論的批判而展開，使我反而對芒福德的《城市發展史》等著作產生了興趣，日後有時間或會精讀。
