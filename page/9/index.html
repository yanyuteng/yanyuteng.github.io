《日本论》笔记

发表于 2015-09-12   |   分类于 Review Sociology

写在随笔前面

戴季陶先生在日本生活了四年，而这本《日本论》撰写发表于1928年，这七十余年前的研究，当可谓对时局把握的极为精准了，于今日日本何以发展如此，颇多启示。
研究日本文化，更为著名的是本尼迪克特的《菊与刀》，但惭愧的是未曾读过，觉得相比于译本，总应该是中国人写的论述更为亲切，便从戴季陶先生的日本论开始读起。
先前在网上简单看了一下《菊与刀》的目录和前几章正文，相比之下，《日本论》应是算不上严格的质性研究，而更多的是作者在日本学习、生活所得的感悟，最后整理成的论集，对于初次深入接触日本文化，应是极好的引导读物，可稍加了解历史上的日本。
但毕竟是七十余年前的读物，今天日本经过战后美国管制、经济腾飞、房产泡沫，再至今日二十一世纪日益全球化，欧美文明的大幅输入，邻国中国的崛起等诸多事件影响，当与书中内容有很多不同，于此书，我更乐于将之作为观察上世纪二十年代之前日本的读物，而谁要说读完了这一本论集，便大概了解日本，我是断然反对的。
书中在叙说实然之后，常有着戴季陶认为的应然，大抵是时代原因，戴季陶作为国民党元老之一，也偶有几个段落在推崇孙文的三民主义。
抛去时代限制，总体而言，《日本论》相对客观的叙述了日本从德川幕府时期至二十世纪三十年代，近代的政治、社会发展变迁，日本文化的一二根源等，但偏向与政治方面，文化方面相对较少。
相比于钱穆的《中国历代政治得失》，字数差不太多，但好读太多。每节不长的薄册子，可于闲暇时间抽空来看，两三日光景的消遣，足矣看完。

《日本人 —— 一个外国人的研究》 笔记

发表于 2015-09-09   |   分类于 Review Sociology

写在随笔前面

蒋百里先生写的小册子极薄，文尾又以仙人老者作结，颇有些志怪小说的感觉。
此论文于1938年出版，为黄埔军校丛书之一，遥想七十余年前，黄埔学生翻读此书，而我现在也捧着这本小书，便多少有些感慨。
内部每节又已划分结构，且读且想，两三个小时便能读完。
其中，当精读第六节与第八节，《军人思想之变迁》与《财政经济》两章，农民军人与垄断财阀政党之关系，以之窥视日本转为军人执政走向之原因。余者泛读，有些了解即可。
每节作者既已有划分，不再赘述，而我胡乱划分了全书除应当精读的六、八两节外的十节，五大方面，供后读者参考。附录第六节与第八节。

一、二两章叙述日本人之文化。
三、四两章叙述日本之崛起。
四、五两章叙述日本之矛盾。
七、九两章详细叙述其矛盾之中，外交、行政之两面背离原因
十一、十二两章，概论日本战争失败之原因。

日记 兴城

发表于 2015-09-06   |   分类于 Travel Diary

15.9.2 启程

上午九点起床，下午四点乘车，同行的还有父母、姐姐、舅舅与舅妈、二姨与二姨夫、姥姥，一家九口。
去兴城，多少是长辈的突然决定，9月1日下午告诉了我，要一起去葫芦岛待上几天。我之前是不知葫芦岛的，便在1号的晚上，简单查了一下，却是发觉葫芦岛的历史颇为有趣。

葫芦岛，旧属锦州，称锦西。
下辖的兴城，与西安、平遥、荆州并称为中国四大古城。绥中，传为吴三桂的出生地。
明清之际，努尔哈赤于此战败，宁锦大捷，努尔哈赤同年忧愤而死，13万大军止步关外。
尔后，袁崇焕为皇太极离间冤死，满清入关，汉明末路。
再至近代，日本战败，百万日侨于此返回日本。内战时，蒋介石于此两次督战，而月余，辽省赤红之旗遍野，一支国民党军队于此逃出。
千年兴衰，关外一城。

《新教伦理与资本主义精神》 笔记

发表于 2015-09-02   |   分类于 Review Sociology

前言

深夜，听着民谣，大不列颠的广场里，写着这篇书名便看起来很严肃的随笔前言。
有些不伦不类。
我读的是阎克文的译本，应是译作里的上乘，但文字仍有些晦涩。我有些固执的觉得，在韦伯的笔下，他的文字，总是一个个藏在阴郁里的诗歌，需要很静很静的时候，才能听到这些文字背后的呼吸。
韦伯多少是个狂想者，倔强而又孤独的整理着，这个世界的宗教。那注释里夹着注释，他的后半生套着前半生。从中国，到印度，再到犹太，诸世界的宗教里，尽是他的经济与社会。
而他的溘然长逝，只剩下一堆未尽的手稿，几千页。学术惋惜，诗歌落寞。
十八岁大概还很难触及韦伯的阴郁，但他天才如迷宫般的构想里，予我智识真正的深度力量，这加尔文宗，那资本主义。
真正的经典，韦伯的孤独。

《中国历代政治得失》随笔

发表于 2015-09-02   |   分类于 Review History

序

想来一切人事为重，忽视革命之功用亦非偏颇，钱穆先生确应重视人事，然苟无乌托邦之批判，则无今日多元思想之冲汇。
但钱穆先生思想确为当时先进，惟因本土之文化，引他山之石，方得立于世界民族之林。遗憾致学组有事，于此离笔。
15.6.4

前言

钱穆先生大才，全书叙汉唐宋明清五朝，列政府职能之流变，政权交付则细化为财经、军事两个制度，阐述中国制度得失，观古示今。

一、制度的考察

对于制度的得失，持以七端。制度与人事相互依托，随人事变动，也自身内部互为配合，必有其前身流变，以历史资料推之，有其得失；看其实际影响，看彼时人的历史意见与当世人的时代意见，历史意见是真依据，不能抹杀；同时考量制度的时代性与地域性；考量制度的特殊性，即考量全部历史之文化意义，而不能剥离政治出来单独探讨。

二、政府职权的分配

考察四维，政府的组织，探讨政府如何组织；科举，政权交由哪些人，又在内部将权力分配如何细分；赋税；国防兵役。

《万历十五年》 笔记

发表于 2015-08-29   |   分类于 Review History

前言

《万历十五年》一书，成于1976年，至今已近四十载。
私以为，是书当为历史学、政治学、社会学，乃至人文社科学生之通识。其横剖明朝一代，以小观大，精妙之处，与人叹焉。
虽争议颇多，或喜或恶。其为专业之书，则论述之中，多参杂黄仁宇之感慨，推论不甚严谨，然此书，终归可为开拓眼界之经典，其表写明朝万历，而隐喻当今世界，古今中西多有对比。诸多未加太多证明之观点，窥视天下进展之看法，予人以极大启发，予我以颇多感触。
此长文为我读书随感，以我感触，将书中一章之内容，胡乱分割为几大部分，稍理其内部结构。
盖读书之时，身处乡村，整理之时，未曾查阅他人感慨。这随笔漫谈，全是混小子的自家看法，我之读书感触，读者笑之可矣，或有错误之处，期读者谅解，赞赏之处，则我之幸也。
如此，《万历十五年》共二十万九千字，字字细读，乃整理成九千余字读书随笔，十八岁之夏，看笔下涓涓，多少有些感动。

日记 延边

发表于 2015-08-27   |   分类于 Travel Diary

15.7.24 延吉

和孟典坐高铁到长春，12:50乘汽车到延吉，妈妈说这是姥爷的故乡，而在我出生的前一年，姥爷去世。
两篇论文尚未写完而出来旅行，大抵是要趁着年轻，多走走看看。我只是个孤独的旅人，喜欢在路上的感觉，6点抵达延吉。

延吉路边上铺的牌子都是中韩双语，但多少是去过海拉尔，看过中俄蒙三语的牌子，在拉萨也见多了中藏双语的牌子，对中韩双语已不是特别的新鲜。
安顿好宾馆后，直奔元奶奶韩国参鸡汤，排骨酱汤、石锅豆腐、大的猪肉饭包，米酒，花费210元，剩了好多，但多少是领略了一下韩餐的感觉。
夜晚的布尔哈通河很漂亮。问了问旁边的路人，布尔哈通河有什么寓意，路人也不知。想了想，笑笑自己，非要问这些典故有什么意义，大概很多事情本就是没有原因。
延吉的夜生活较哈尔滨来的丰富，路边很多的酒吧和啤酒屋。和孟典在酒吧坐了坐，喝了几口冰川，延吉的夜色温柔。
江边有很多朝鲜族人漫步，青年广场之下，汉语与韩语间杂，很多美女，一些小贩，小城市的生活无端让我安心。

日记 雅尔塞

发表于 2015-08-26   |   分类于 Travel Diary

15.8.8 支教启程

支教启程。
早上六点多起床，6:50到哈工大一区的校训石，8点多乘公交到哈西，9:22乘上火车，1点多到齐齐哈尔，出了火车站，转乘大巴1个多小时，到达梅里斯区支教的雅尔塞小学。
梅里斯区楼房不超过七层，感觉更多像是城乡结合部，雅尔塞小学在梅里斯区的边缘，一条马路的小镇。
小学操场虽然大些，但很简陋，都是沙泥地。校园后是一片无垠农田，打开走廊窗户，草香扑面。小学如厕是东南角的旱厕，多年未用旱厕后喷薄而出，高空抛落，忆来颇爽。
来之前听说我们一切用水都是靠一口井水，的确如此，但并不是我们想象的井。井边是一圈围起的矮砖，井口被泥土封死，一根水管孤零零立在中央，泥土下是一个水泵，插上电后，从地下滋滋汲水。井的智慧，在干旱无水或离水较远处，省去多少麻烦。井水颇凉，我们将近二十日的吃喝全部赖此。
与来之前设想不同，齐齐哈尔的支教点是第一年开设，与山东情况不同，我们的受众可能更多的把我们当做夏令营。但我们此行三周，所要讲授知识，本就包罗万象，天文地理，历史神话，所求更多是开拓少年之眼界，世界之大，值得我们一生探索，我们亦勿忘来之初心。
安顿好后去买菜和生活用品，晚上切土豆炒土豆，很快见底，简单开会后各自散去。
雅尔塞的夜晚没有灯，假设丢掉手电，真是寸步难行。几个男生光着膀子，用一个小台灯，用刚抽的井水洗澡。
