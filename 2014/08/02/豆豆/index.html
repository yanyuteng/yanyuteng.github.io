豆豆

发表于 2014-08-02   |   分类于 Prose   |     |   本文總閱讀量 次

7月26日，到今天已是一周过去。我写这篇日志，不是为了他人的怜悯同情，只是想把心中抑郁慢慢打出来，觉得我矫情的人大可不看，觉得我伤心的人也不必安慰。我只是想把一些事情诉诸文字。
没有养过狗的人大概不会理解这种丢狗的心情，每当我去问路人，他们总会说：“狗嘛，丢就丢了。”他们不懂。

或许狗最廉价的地方就在于，主人是他的一切，而它只是主人的部分。但是，狗最高贵的地方也在于，它虽只是主人的部分，但主人就是它的一切。
当一只巴掌大的小狗，到快6岁的壮年，它陪我度过我18岁的三分之一，时间流过，现在窗外只是一片阴天，它已不在我的身边。
我将再也不会见到大厅里，阳光下它悠然的晒太阳。我将再也不会为阳台的狗屎狗尿发愁。我将再也不会在回家时迎来蹦蹦跳跳。我将再也不会抚摸它的肚皮。我将再也不会和它嬉闹，我将再也不会在夜深人静时，被它默默注视，默默陪伴。
它留给我的只是几张干涩的照片，杳无音信的空白，以及深深地自责与思念。

它在一生中最好的日子陪伴了我，陪我度过初中、高中，初四、高三，我终于长大，而它却老了。
我不知道它在外面是否会流浪，会在下雨打雷时惊慌失措。我不知道它是否会被人收养，在一个小卫生间里，被人圈起。我不知道它是否还能再欢快的吠叫，被人轻轻的抚摸。我甚至不知道，它是否还尚在人间。
在它丢失那天，它还没有吃东西，饿着一天的肚子，在黑夜里去寻找我们。当我在外地得知它走失时，我在宾馆的房间里抱头痛哭，坐上第二天的飞机回哈时，已是它走丢的第四天。下了飞机，把行李放在家里，便坐车去它丢失的地方，没有人见过，仿佛人间蒸发。

贴吧论坛，报纸告示，却没有电话打来。
佛家讲缘起缘落，自有定数，可能我与豆豆缘分尽了。妈妈安慰我说，小豆豆或许是陪另一个小孩的成长，或许是陪另一个老人的晚年 。它那么可爱，必定是个好人家的。
或许人毫无希望时就会寄托神灵，我不会看易经，却自己起卦，没敢告诉爸妈，卦象都是不能寻回。只是强撑笑脸，妈，别着急了，大吉的。
它对我的意义，绝不仅仅是狗，是我的成长，在我青春躁动日子里，默默舔舐我的双手，在我高三求学时，默默趴伏我的脚边，我认识很多人不及六年，六年前的很多人也早已陌路，只有它，默默在我身边，陪我度过青春，如今也不知去向。
很多时候，就像韩寒新拍的电影，我还没看，也没有心思，只是看到他的一句台词。“每一次告别，最好用力一点。多说一句，可能是最后一句。多看一眼，可能是最后一眼。”没有失去，就永远不懂珍惜。很多平常的东西，也只有失去，才知道会是那么痛彻心扉。
找了一周，希望渺茫，我已不抱希望能够寻回它，只是希望它被人抱走，有一个更好的家，不会流浪街头。
想再打些什么，可是却终究发现，文字也太过苍白。

整理博客时又看到这篇文章，豆豆已经走失近2年了，时间好快，而我愈发悲伤，我清楚的记得当时的悲怮，即便如今依旧隐隐作痛，而文字苍白，不知该写些什么。
16.6.1
闫誉腾 高雄
