哈六中记

发表于 2014-06-22   |   分类于 Prose   |     |   本文總閱讀量 次

余尝闻：“三六九附，皆名校也。”余求学于六中，已三载矣。而今将远离，戚然，乃作此文以自怀。
六中坐于香坊之公滨，盖六中之美，如倩女之靧面，而髻鬟之始掠也，妍媚娟然，其间青松劲柏，四时常青。若夫鸢飞戾天，经纶事务者，毕悉心忘返，其清净若此，余常私拟作欲界之仙都。

早春时节，天朗气清，惠风和畅，楼舍之间，小径相属，径边桃花映面，绿草欣然，及主道，两侧遍植杨柳，柳条将舒未舒，柔俏披风，柳河东尝趋小石潭曰：“青树翠蔓，蒙洛摇缀，参差披拂。”盖六中亦如是也。趋两主楼之间，则耳边有庠生诵读，飞鸟轻鸣。至于桃花落后，花残飘零，暮春之伤，与人叹焉。

及夏日，两主楼之间有一小池，池中游鱼细石，直视无碍，池边亭林石凳，颇俱古之遗风。七月流火，余常偕友来此，俯读仰唱，不亦快哉。

比至秋冬之时，天云俱缈，秋风呜咽。若逢雨雪霏霏。则雾凇沆砀，余乐与同窗嬉游雨中，听周遭噼啪作响，而余闲庭信步，虽或感风寒，未曾舍弃。而每逢大雪，雪披万物，操场深数尺之时，余则卧躺其中，此间乐难为外人道也。

至于食堂之饭菜，诚单调枯燥，然，此为六中之味，而他处难得。平日嬉笑怒骂间，同窗之情，诚为手足之情。授余书之众先生，皆德隆望尊，三载课堂，不辞劳苦，余所学者，非为格物之理，亦为处世之道，舍众先生，余难成学业。

笔至此，食堂之饭菜，同窗之情谊，师长之教诲，历历在目，书短意长，难以言尽。三年委短，若白驹过隙，余已成年，思之，余青春年华，乃尽藏于六中之一石一草，尽融于六中之一花一木。今记六中之事，亦记余之三载青春，六中代代不舍，廿年后，六中依旧，则余之青春依旧。

余亦常念六中之先辈，风雨传承百载，其间有民国初建，伪满经营，惟六中之代代不舍，余乃能求学于此，每怀之，亦荣为一六中之人。六中校训：“崇德求是。”乃余身为六中之人一生所当秉持，所当发扬。

捌月后，余将远离，适六中九十华诞，憾不能亲与盛会，惟愿六中之青春不逝，虽百载老校，亦生机勃勃。

2014.6.22
闫誉腾 哈尔滨

后记
昨天和明理学弟吃饭，猛然发觉今年九月即已大二，而高中仿若昨日。今晨偶然翻到去年六月高考结束所写之文，十七岁之缅怀，也是有趣。更为有趣的是，去年文尾所哀，高考或将录取到异乡大学，而不能参与母校的九十华诞，成了多想。却是录了本地，也参与了校庆。
人生阴差阳错大抵如是，而叠叠巧合之下，多少让人感到了命运的可爱。如是，看以前的文章，写新的感触，一生平淡而过，未尝不好。
2015.8.29 11:00
闫誉腾 哈尔滨
