R語言基礎 數據集結構與讀取、隨機數

发表于 2016-06-03   |   分类于 Program   |     |   本文總閱讀量 次

一、數據集結構

（一）、向量 Vector

1、含義
向量就是一維數組,包含數字、字符、邏輯語句（TRUE&FALSE）,用c這個function組成

2、操作
a = c(1, 2, 5, 3, 6, -2, 4)
b = c(“one”, “two”, “three”)
c = c(TRUE, TRUE, TRUE, FALSE, TRUE, FALSE)

3、讀取
#賦值
v＝c(1,2,3,4,5,6,7)
#返回第2個,第4個,第3個
v[c(2,4,3)]
#返回第二個到第四個
v[2: 4]
#刪除第二個元素後的元素集
v[-2]
#刪除第二個到第四個元素
v[-2: -4]
#返回v中所有小於3的元素
v[v<3]
#找到所有等於3的元素, =賦值, ==相等（出現的是序號）
which(v== 3)
#返回第一個最大值
which.max(v)
#返回第一個最小值
which.min(v)

（二）、矩陣 二維數組

1、含義
兩維數組,matrix這個function組成

2、操作
# 5行4列 nrow=5, ncol=4
#按行來填充數據
x = matrix(1: 20, nrow=5, ncol=4, byrow=TRUE)
#按列來填充數據,系統默認
y = matrix(1: 20, nrow=5, ncol=4, byrow=FALSE)

3、矩陣內部數據簡單操作
#返回第二行
x[2,]
#返回第二列
x[,2]
#返回特定值,第1行第4列的數
x[1,4]
#其他例子
x[2,c(2,4)]
x[3: 5,2]

#進一步定義matrix行和列的名字,要與其行列一致,否則會報錯
rnames = c(“apple”,”banana”,”orange”,”melon”,”corn”)
cnames = c(“cat”,”dog”,”bird”,”pig”)
x = matrix(1: 20, nrow=5, ncol=4, byrow=TRUE)
rownames(x) = rnames
colnames(x) = cnames
x
#簡化
x = matrix(1: 20, nrow=5, ncol=4, byrow=TRUE)
rownames(x)=c(“apple”,”banana”,”orange”,”melon”,”corn”)
colnames(x)=c(“cat”,”dog”,”bird”,”pig”)
x

（三）、多維矩陣 Array

#Array 數據來源；維度的長度,第2維幾個,第3維是幾個；dimnames默認不用設置,設置要用list處理
#展示第二維數組,三維以上沒有辦法很好顯示,所有C、D組合
dim1 = c(“A1”, “A2”)
dim2 = c(“B1”, “B2”, “B3”)
dim3 = c(“C1”, “C2”, “C3”, “C4”)
dim4 = c(“D1”, “D2”, “D3”)
z = array(1: 72, c(2, 3, 4, 3), dimnames=list(dim1, dim2, dim3, dim4))
z
#返回操作類似于matrix,返回第一維1,第二維2,第三維3,所有第四維
z[1,2,3,]

（四）、data.frame 使用

#一般是數量較多的大矩陣 兩維
1、練習輸入
patientID = c(1, 2, 3, 4)
age = c(25, 34, 28, 52)
diabetes = c(“Type1”, “Type2”, “Type1”, “Type1”)
status = c(“Poor”, “Improved”, “Excellent”, “Poor”)
patientdata = data.frame(patientID, age, diabetes, status)
patientdata

2、操作
#取出前兩列內容
patientdata[,1: 2]
或patientdata[1: 2]
#取出前兩行內容
patientdata[1: 2,]
#取出第1行、第3行,第1列到第3列的內容
patientdata[c(1,3),1: 3]
3、attach detach使用
#R內置的mtcars數據,車油耗與速度重量之間關係
attach(mtcars)
mtcars
mpg
detach(mtcars)

（五）、list使用

#不受維度、數據長度等限制,比如一個62行,一個5行,用data.frame會報錯,list則無視這些,但相應在取出部分會稍微麻煩一點

#查詢第一部分內容
mylist[1]

1、前置的數據準備
x = matrix(1: 20, nrow=5, ncol=4, byrow=TRUE)
swim = read.csv(“http://www.macalester.edu/~kaplan/ISM/datasets/swim100m.csv“)
mylist=list(swim,x)
mylist[[2]][2]

2、實際如何取出
#返回第二個list的第二個元素,默認是縱向的
mylist[[2]][2]
#返回第二個list的第三列的前兩個元素
mylist[[2]][1: 2,3]
#返回第二個list前兩行內容
mylist[[2]][1: 2,]

二、讀取數據

#直接讀取
data1= read.csv(file=”~/desktop/xxx”)
data2=read.table(file=”~/desktop/xxx”)
#列名稱變成變量名稱使用
attach(data1)
#輸入讀取（適於簡單樣本）
x <- scan()
#再把excel數據複製回車

三、隨機數

（一）、隨機數的生成

1、均一分部的隨機數（偽隨機數）
#set.seed(xxx)是要設置一組固定的偽隨機數,便於科學重複實驗,為runif或rnorm前置
set.seed(250)
a = runif(3,min=0,max=100)

2、正態分佈的隨機數
#默認均值mean=0, 標準差sd=1
rnorm(3, mean=0, sd=1)

set.seed(123)
x=rnorm(100,mean=100,sd=10)
set.seed(234)
y=rnorm(100,mean=100,sd=10)

（二）、小數點取捨

#小數捨去,取整數
floor(a)
#無論小數多少,都進1
ceiling(a)
#保留a的4位小數
round(a,4)
