（二）、區間估計

1、單總體特征值

（1）、μ
A 大樣本 n>=30 ~N
B 小樣本 ~T
（2）、π
A 大樣本 np>=5、n(1-p)>=5、n>=30 ~N
B小樣本 「不能估計」

2、兩總體特征值

（1）、均值差
A 大樣本
條件 n1>30；n2>30；
服從分佈 ~N分佈
B 小樣本
條件 σ1^2=σ2^2; 兩總體~N
服從分佈 ~T分佈
（2）、頻率差 P1-P2
A 大樣本
條件 n1p1>=5; n1(1-p1)>=5; n2p2>=5; n2(1-p2)>=5
服從分佈 ~N
B 小樣本不能使用

一、判斷語句

1.基於計算機邏輯 Three-valued Logic

1）大小號的使用

大於 >
大於等於 >=
小於 <
小於等於 <=

2）等號的使用

等於 ==
賦值 =
# 賦值的另一種嚴謹寫法是 ->

不等於 !=
x且y x&y
x或y x|y

# 範例 賦值x為1,2,3,4
x = c(1,2,3,4)

（三）、switch語句

feelings = c(“love”, “bigface”)

四、自定義的Function Uesr-define function

#function(x,x,x) 內部需要幾個參數，return根據情況寫，這裡是隨便寫的函數
myfunction = function(x,a,b,c){
return(a*sin(x)^2-b*x+c)
}

#畫出函數，curve 的第一個空格必須是一個函數，xlimit設置了起點和終點
curve(myfunction(x,20,3,4),xlim=c(1,20))
curve(exp(x),xlim=c(1,20))

#練習定義一個實際的function
myfeeling = function(x){
    for(i in x){
        print(
        switch(i,
        happy = “I am glad you are happy”,
        afraid = “There is nothing to fear”,
        sad = “Cheer up”,
        angry = “Calm down now”
                     )
                )
           }
}
feelings=c(“sad”,”afraid”)
myfeeling(feelings)

六、補充 factor

#好似標記，給向量、矩陣標記

1、自動計算

#rep循環
factor=factor( rep(c(1:3), times=5) )

#0-100選15個數字的隨機樣本
x=sample(100,15)

#x用factor來標記的平均數
tapply(x,factor,mean)

2、手動計算驗證

#看一下x里 哪些數被標記了
rbind(x,factor)

#手動求一下，合成為2行，查看所有被標記為2的平均數
boo =rbind(x,factor)[2,]==2

#返回所有是TRUE的序列號
which(boo)

#查看返回的那部分和之前的是不是一樣的
rbind(x,factor)[1,which(boo)]

#查看和/長度＝平均數
sum(brind(x,factor)[1,which(boo)])/length(which(boo))

第三节 數據集結構與讀取、隨機數

2016-06-03 15:15:00

一、數據集結構

（一）、向量 Vector

1、含義
向量就是一維數組,包含數字、字符、邏輯語句（TRUE&FALSE）,用c這個function組成

2、操作
a = c(1, 2, 5, 3, 6, -2, 4)
b = c(“one”, “two”, “three”)
c = c(TRUE, TRUE, TRUE, FALSE, TRUE, FALSE)

3、讀取
#賦值
v＝c(1,2,3,4,5,6,7)
#返回第2個,第4個,第3個
v[c(2,4,3)]
#返回第二個到第四個
v[2: 4]
#刪除第二個元素後的元素集
v[-2]
#刪除第二個到第四個元素
v[-2: -4]
#返回v中所有小於3的元素
v[v<3]
#找到所有等於3的元素, =賦值, ==相等（出現的是序號）
which(v== 3)
#返回第一個最大值
which.max(v)
#返回第一個最小值
which.min(v)

（二）、矩陣 二維數組

1、含義
兩維數組,matrix這個function組成

2、操作
# 5行4列 nrow=5, ncol=4
#按行來填充數據
x = matrix(1: 20, nrow=5, ncol=4, byrow=TRUE)
#按列來填充數據,系統默認
y = matrix(1: 20, nrow=5, ncol=4, byrow=FALSE)

3、矩陣內部數據簡單操作
#返回第二行
x[2,]
#返回第二列
x[,2]
#返回特定值,第1行第4列的數
x[1,4]
#其他例子
x[2,c(2,4)]
x[3: 5,2]

#進一步定義matrix行和列的名字,要與其行列一致,否則會報錯
rnames = c(“apple”,”banana”,”orange”,”melon”,”corn”)
cnames = c(“cat”,”dog”,”bird”,”pig”)
x = matrix(1: 20, nrow=5, ncol=4, byrow=TRUE)
rownames(x) = rnames
colnames(x) = cnames
x
#簡化
x = matrix(1: 20, nrow=5, ncol=4, byrow=TRUE)
rownames(x)=c(“apple”,”banana”,”orange”,”melon”,”corn”)
colnames(x)=c(“cat”,”dog”,”bird”,”pig”)
x

（三）、多維矩陣 Array

#Array 數據來源；維度的長度,第2維幾個,第3維是幾個；dimnames默認不用設置,設置要用list處理
#展示第二維數組,三維以上沒有辦法很好顯示,所有C、D組合
dim1 = c(“A1”, “A2”)
dim2 = c(“B1”, “B2”, “B3”)
dim3 = c(“C1”, “C2”, “C3”, “C4”)
dim4 = c(“D1”, “D2”, “D3”)
z = array(1: 72, c(2, 3, 4, 3), dimnames=list(dim1, dim2, dim3, dim4))
z
#返回操作類似于matrix,返回第一維1,第二維2,第三維3,所有第四維
z[1,2,3,]

（四）、data.frame 使用

#一般是數量較多的大矩陣 兩維
1、練習輸入
patientID = c(1, 2, 3, 4)
age = c(25, 34, 28, 52)
diabetes = c(“Type1”, “Type2”, “Type1”, “Type1”)
status = c(“Poor”, “Improved”, “Excellent”, “Poor”)
patientdata = data.frame(patientID, age, diabetes, status)
patientdata

2、操作
#取出前兩列內容
patientdata[,1: 2]
或patientdata[1: 2]
#取出前兩行內容
patientdata[1: 2,]
#取出第1行、第3行,第1列到第3列的內容
patientdata[c(1,3),1: 3]
3、attach detach使用
#R內置的mtcars數據,車油耗與速度重量之間關係
attach(mtcars)
mtcars
mpg
detach(mtcars)

（五）、list使用

#不受維度、數據長度等限制,比如一個62行,一個5行,用data.frame會報錯,list則無視這些,但相應在取出部分會稍微麻煩一點

#查詢第一部分內容
mylist[1]

1、前置的數據準備
x = matrix(1: 20, nrow=5, ncol=4, byrow=TRUE)
swim = read.csv(“http://www.macalester.edu/~kaplan/ISM/datasets/swim100m.csv“)
mylist=list(swim,x)
mylist[[2]][2]

2、實際如何取出
#返回第二個list的第二個元素,默認是縱向的
mylist[[2]][2]
#返回第二個list的第三列的前兩個元素
mylist[[2]][1: 2,3]
#返回第二個list前兩行內容
mylist[[2]][1: 2,]

三、隨機數

（一）、隨機數的生成

1、均一分部的隨機數（偽隨機數）
#set.seed(xxx)是要設置一組固定的偽隨機數,便於科學重複實驗,為runif或rnorm前置
set.seed(250)
a = runif(3,min=0,max=100)

2、正態分佈的隨機數
#默認均值mean=0, 標準差sd=1
rnorm(3, mean=0, sd=1)

x=rnorm(100,mean=100,sd=10)
set.seed(234)
y=rnorm(100,mean=100,sd=10)

round(a,4)
