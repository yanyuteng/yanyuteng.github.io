一、判斷語句

1.基於計算機邏輯 Three-valued Logic

1）大小號的使用

大於 >
大於等於 >=
小於 <
小於等於 <=

2）等號的使用

等於 ==
賦值 =
# 賦值的另一種嚴謹寫法是 ->

不等於 !=
x且y x&y
x或y x|y

# 範例 賦值x為1,2,3,4
x = c(1,2,3,4)

二、循環語句 Control Flow

（一）、for loop

for(i in 1:10){
print(i)
}

三、條件語句 If Else

（一）、If語句

# == 意義是「等於」

i=1
if(i == 1){
print(“hello world!”)
}

（二）、if else語句

# else不是必須的

i=2
if(i == 1){
print(“hello world!”)
}
else{
print(“Goodbye World!”)
}

# elseif R語言未內置，實際是switch，

（三）、switch語句

# switch
# switch(expression, cnoditions)

feelings = c(“love”, “bigface”)
for (i in feelings){
print(
  switch(i,
    love = “hhh, coz I’m too handsome”,
    like = “yeah, I konw u like my sociology blog”,
    hate = “Never mind, u will love me finally”,
    bigface = “hhh, u are so cute”
             )
      )
}

# 輸出的是”hhh, coz I’m too handsome” “hhh, u are so cute”
# 你也可以嘗試用中文寫一個把妹的段子 :)

四、自定義的Function Uesr-define function

#function(x,x,x) 內部需要幾個參數，return根據情況寫，這裡是隨便寫的函數
myfunction = function(x,a,b,c){
return(a*sin(x)^2-b*x+c)
}

#畫出函數，curve 的第一個空格必須是一個函數，xlimit設置了起點和終點
curve(myfunction(x,20,3,4),xlim=c(1,20))
curve(exp(x),xlim=c(1,20))

#練習定義一個實際的function
myfeeling = function(x){
    for(i in x){
        print(
        switch(i,
        happy = “I am glad you are happy”,
        afraid = “There is nothing to fear”,
        sad = “Cheer up”,
        angry = “Calm down now”
                     )
                )
           }
}
feelings=c(“sad”,”afraid”)
myfeeling(feelings)

六、補充 factor

#好似標記，給向量、矩陣標記

1、自動計算

#rep循環
factor=factor( rep(c(1:3), times=5) )

#0-100選15個數字的隨機樣本
x=sample(100,15)

#x用factor來標記的平均數
tapply(x,factor,mean)

2、手動計算驗證

#看一下x里 哪些數被標記了
rbind(x,factor)

#手動求一下，合成為2行，查看所有被標記為2的平均數
boo =rbind(x,factor)[2,]==2

#返回所有是TRUE的序列號
which(boo)

#查看返回的那部分和之前的是不是一樣的
rbind(x,factor)[1,which(boo)]

#查看和/長度＝平均數
sum(brind(x,factor)[1,which(boo)])/length(which(boo))
