西子灣隨筆

发表于 2016-02-28   |   分类于 Prose   |     |   本文總閱讀量 次

2月24日那天，我坐在西子灣的堤壩上。
以前看到大海常會感歎人類于自然的渺小，可是那個下午，卻猛然覺得，人類于幾千年歷史中的偉岸，縱使與蓋亞相比小如螻蟻，而終歸已是浮萍在浩大宇宙中的驚濤拍岸，其史無前例的巨響，激勵著一輩一輩。
那時起風、大浪，漁船回岸。
大霧里，耳機里是岸部真明的奇跡之山，我背靠的是高山，面朝的是大海，充滿迷夢的八十年代里，中山大學立校以山海胸襟，這是何等的氣魄。
又是何等的難辦。
我總是不自覺的將台灣和日本相近，令人心折的物哀文化，以及類武士道精神的豪情，充滿矛盾的張力里，孕育了大島的溫情。

現在我坐在五嶺的村莊里。
手邊有一摞文獻未讀，寢室也因為窗外的高樹而頗為昏暗。
我無數次夢想過逃離，就像里爾克的漂泊，“一個人只有在第二故鄉，才能檢視自己靈魂的強度和靈魂的承載力。”
此行半年亦然。
我相信當人開始認真的沉思，就不免將要陷入深深的無力中，不單是于個人命運在巨大社會中的起起伏伏、不由自己，亦有其他各人的不同悲憫。
上課一周，台灣教育與大陸並無二致，本以為的脫嵌，也只是嵌入另一個牢籠里。
可是大海這麼無悲無喜，個體人生的苦痛在千萬年間也仿佛變成了笑談，大概在天地看來，所有的監獄都是自己築造。
這世間的諸多枷鎖，大概都是自己的建構，道德也好、傳統也罷。當後現代開始解構，是混亂與真正啟蒙的開始。
我將無法預見真正自由來臨之時，大抵莫爾之構想過於艱難，可社會理想主義終歸支撐一代一代孤獨的潛行者，也會波及一代一代之普羅大眾。
余光中有一句話可以拓展，“面向大海，就擁抱了世界。”這肉體的漂泊終歸會停止，而我長眠之時，心靈的漂泊永無終止，無數大海之廣闊，足矣人類之不盡探索。
海浪愈大，我心愈靜。

2016.2.28
闫誉腾 高雄
